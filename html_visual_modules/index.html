Integration Guide

Quick Start

Each module is a self-contained HTML file that can be easily integrated into your Math Blaster application:

// Include shared utilities <link rel="stylesheet" href="shared-styles.css"> <script src="shared-utils.js"></script> // Include specific module <script> const visualizer = new NumberLineVisualizer('canvasId'); visualizer.loadProblem(problemData); </script>

Data Format

All modules expect problem data in a standardized format from the Python backend:

{ "type": "number_line|array|area_model|quotative", "operation": "addition|subtraction|multiplication|division", "problem_statement": "5 + 3 = ?", "visual_data": { /* module-specific data */ }, "answer": 8 }

Key Features

  • Responsive design that works on all screen sizes
  • Interactive animations and visual feedback
  • Answer validation with immediate feedback
  • Hint systems and step-by-step guidance
  • Consistent Math Blaster visual theme
  • Modular architecture for easy customization
  • Canvas-based rendering for smooth graphics
  • Accessibility features and keyboard navigation

Customization

All modules use CSS custom properties and can be easily themed:

:root { --primary-color: #00ff00; --secondary-color: #00cc00; --background-color: #0a0a0a; --text-color: #00ff00; }

Backend Integration

Connect with the Python visual modules to generate problem data:

# Python backend from visual_modules import NumberLineModule module = NumberLineModule() problem = module.generate_addition_problem() visual_data = module.get_visual_description(problem) # Send to frontend return json.dumps(visual_data)
